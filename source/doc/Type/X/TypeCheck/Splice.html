class X::TypeCheck::Splice is X::TypeCheck does X::Comp { }

Compile time error thrown when a Macro or an unquote/hole in a quasi quote does not return an AST.

For example

macro a { 'foo' }
say a

dies with

===SORRY!===
Type check failed in macro application; expected AST but got Str

To fix, instead write

macro a {
    quasi { 'foo' }
}
say a;              # foo
