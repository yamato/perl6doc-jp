class X::Export::NameClash

  1. Methods
    1. method symbol
class X::Export::NameClash does X::Comp { }

Compile time error thrown when a symbol is exported twice.

For example

sub f() is export { };
{
    sub f() is export { }
}

dies with

===SORRY!===
A symbol '&f' has already been exported

Methods

method symbol

Returns the symbol that is exported twice.
